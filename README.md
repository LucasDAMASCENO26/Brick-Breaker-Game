# Brick Breaker Game

A classic Brick Breaker game implemented using Pygame. The game involves controlling a paddle to bounce a ball and break all the bricks on the screen.

## Features

- Real-time gameplay with smooth graphics.
- Start screen with game title and creator name.
- Game over and victory screens with score display.
- Keyboard controls for paddle movement.
